# ⚠️ IMPORTANT: Complete These Firebase Steps

The `.env` file is configured, but you need to enable two services in Firebase Console:

## 🔥 Step 1: Enable Google Authentication (REQUIRED)

1. Go to: https://console.firebase.google.com/project/focus-e7252/authentication
2. Click **"Get started"** button
3. Click on **"Google"** in the list of providers
4. Toggle the **Enable** switch to ON
5. Select your email (noorulilham3@gmail.com) from the dropdown
6. Click **"Save"**

## 🔥 Step 2: Create Firestore Database (REQUIRED)

1. Go to: https://console.firebase.google.com/project/focus-e7252/firestore
2. Click **"Create database"** button
3. Select **"Start in test mode"** (radio button)
4. Click **"Next"**
5. Choose your location (asia-south1 or closest to you)
6. Click **"Enable"**
7. Wait for database creation (~30 seconds)

## ✅ After Completing Both Steps:

The app should work! The dev server is trying to start but needs these services enabled.

**Once you've done both steps above, the app will be fully functional at http://localhost:3000**
